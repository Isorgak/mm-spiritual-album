# Mezgeb Mezmur тАУ Spiritual Songs Archive Website

A modern, responsive website dedicated to preserving and sharing spiritual songs used in worship by churches and choirs around the world.

---

## Project Description

This project is a website created to help preserve and share spiritual songs used in worship. Many beautiful songs exist only in memory or scattered handwritten notes, passed down through generations in churches and communities. The goal of **Mezgeb Mezmur** is to collect, organize, and make these songs accessible for churches, choirs, and believers worldwide.

The website showcases **"I Trust in the Lord"** тАУ a spiritual music album performed by the **St. Joseph La Salle Keren Choir** from Eritrea, featuring 11 sacred songs of faith, hope, and devotion.

---

## About the Project

### Our Mission

Mezgeb Mezmur (which means "Song Collection" in Tigrinya) began in **2013** when a small group of volunteers recognized that many spiritual songs were at risk of being lost forever. Over the past **12 years**, our volunteer network has expanded across regions, bringing together people who share a passion for preserving spiritual heritage.

### What We Do

- **Preserve spiritual songs** тАУ Document and archive songs that exist only in memory or handwritten notes
- **Support church choirs** тАУ Provide resources for worship teams and choir leaders
- **Organize lyrics and recordings** тАУ Create a structured collection of worship music
- **Build a digital archive** тАУ Develop a platform that makes songs accessible to believers worldwide
- **Help future generations** тАУ Ensure that spiritual songs are not lost to time

## Technology Used

This website is built as a **static website** using only core web technologies:

- **HTML5** тАУ Semantic markup for structure and content
- **CSS3** тАУ Modern styling with Tailwind CSS (via CDN)
- **Vanilla JavaScript** тАУ Interactive features without frameworks

### Key Features

- **Responsive Design** тАУ Fully responsive layout that works on desktop, tablet, and mobile devices
- **Embedded YouTube Video** тАУ Modal popup video player for the album trailer
- **HTML5 Audio Player** тАУ Custom audio player with play/pause controls and progress bar for song samples
- **Smooth Animations** тАУ Scroll-triggered reveal animations for a polished user experience
- **Lightbox Gallery** тАУ Image viewer for browsing album artwork and photos

### No JavaScript Frameworks

This project intentionally uses **no JavaScript frameworks** such as React, Vue, or Angular. It is built with pure vanilla JavaScript to keep the codebase simple, lightweight, and easy to maintain.
